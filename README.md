### Prerequisite
```sh
$ git clone this project
$ docker-compose up
```


# Loyalty Backend (requires go1.9.1 or above)

```sh
$ cd backend && go build
$ ./backend (server is runningn on localhost:6061)
```

# Loyalty Frontent

```sh
$ cd frontent && yarn install
$ yarn start
$ Navigate to http://localhost:4350
```
